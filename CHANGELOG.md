# Changelog

All notable changes to `filament-locationpickr-field` will be documented in this file.

## 3.0.0 - 2024-XX-XX

-   Updated the plugin to support `Filament V4`
-   Updated PHP requirement to ^8.2 (supports PHP 8.2, 8.3, and 8.4)
-   Updated Laravel requirement to ^11.0 || ^12.0 (supports Laravel 11 and 12)
-   Added `filament/infolists` dependency
-   Fixed missing `JsonException` import in Infolists component
-   Fixed missing `precision` property in Infolists component
